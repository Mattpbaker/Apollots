# Apollo Training Session Presentation

A clean, modern presentation website for team training sessions with interactive features including timers, task tracking, and role management.

## Features

- **Slide-based navigation** with arrow keys and on-screen buttons
- **Per-section timers** with visual warnings
- **Interactive task updates** with status tracking
- **Role management** with assignment tracking
- **EDI Policy review** sections
- **Marketing Clinic** with interactive forms
- **Timeline visualization** for upcoming tasks

## Setup

Simply open `index.html` in a web browser to use the presentation.

## Deployment

This project can be deployed to Vercel or any static hosting service.
